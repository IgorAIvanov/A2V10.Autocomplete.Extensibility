# A2V10.Autocomplete.Extensibility

Visual Studio extension based on `VisualStudio.Extensibility` that adds autocomplete support for A2V10 XAML files through a bundled language server.

## Solution structure

- `src/A1V10.Xaml.Autocomlete` — Visual Studio extension package
- `src/A2V10.Xaml.LanguageServer` — XAML language server
- `src/A2V10.Xaml.Core` — shared core logic
- `src/A2V10.Xaml.Reflection` — metadata and reflection support
- `tests/A2V10.Xaml.Tests` — automated tests
